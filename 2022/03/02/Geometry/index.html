Games101：Geometry

Introduction

1.Representation

1.1 Implicit

Based on classifying points (Point satisfy specified relationship)
[Example]

Sphere : All points in 3D, where
$$
x^2+y^2+z^2=1
$$

Generally:


$$
f(x,y,z)=0
$$
问题：不直观。对于已知表面方程，判定其形状困难，但是对于具体点判断与该表面的位置关系是容易的。

1.1.1 Algebraic Surfaces

1.1.2 Constructive Solid Geometry (CSG)

Combine implicit geometry via Boolean operations.

通过基本几何的基本运算形成新的几何。

应用广泛，应用在各种建模软件中。

1.1.3 Distance Functions

Definition: Giving minimum distance (could be signed) from anywhere to object.

空间中任一点到所要表述的几何形体表面的最小距离。(外部+，内部-)

Using Distance Functions to blend surfaces together.

先将距离函数融合，然后通过融合之后的距离函数恢复图形。

moving boundary

直接对图形做线性融合效果不好，对距离函数做融合相当于对边界做融合，效果好。（边界： SDF=0）

1.1.4 Level Set Methods (水平集)

存储DF=0的点，构成水平集。

1.1.5 Fractals （分形）

自相似。

某些复杂的分形几何渲染时会引起走样。

1.2 Explicit

方式：given directly / parameter mapping


$$
f:R^2->R^3
$$

$$
(u,v)->(x,y,z)
$$

[Example]
$$
f(u,v)= ((2+cosu)cosv, (2+cosu)sinv, sinu)
$$
问题：难以判断具体点和表面的位置关系

1.3 Conclusion

No “Best” Representation – Geometry is Hard!

Best Representation depends on the task.

Curves

1.Explicit Representations

1.1 Point Cloud

表示方法：“一堆点”

点云密度不高时，很难画出效果好的图形（Difficult to draw in under-sampled regions)

1.2 Polygon Mesh (多边形网)

Most common

1.3 obj

全称：Wavefront object file Format

v 8个顶点的xyz坐标
vn 6种不同的法线（实际自动建模产生了冗余）
vt 每个面四个点纹理坐标，共12个
f 定义点构成三角形的连接关系（图中依次指代 点坐标/纹理坐标/法线 （均为序号））

1.4 Bézier Curves （贝塞尔曲线）

四个控制点P0,P1,P2,P3定义曲线，起点P0,终点P3,起始处切线方向P0P1，终点处切线方向P2P3，乘以常系数3.

控制点未必一定要求曲线经过！

1.4.1 绘制

de Casteljau Algorithm 求解

一种递归算法，反复进行线性插值。

控制点可以扩展到三维空间。

贝塞尔曲线性质

注意：

3.对仿射变换具有性质：对顶点做仿射变换，然后根据变换后的顶点画曲线；和直接对原曲线上的每一个点做变换得到新曲线的结果相同。但是对投影变换没有这个性质。

4.凸包性质：画出的贝塞尔曲线一定在控制点组成的凸包内。

1.4.2 逐段贝塞尔曲线

一般用四个控制点控制的三阶贝塞尔曲线组成整个曲线。

1.4.3 连续性


$$
C0连续：两段曲线接触点数值连续
$$

an=b0

$$
C1连续：两段曲线数值连续，一阶导数连续

1.5 其他曲线

1.5.1 splines（样条）

简单定义：一条可控的曲线

B-splines（basis splines 基函数样条）

具有局部性：可以对曲线的某一部分进行修改而不影响整体，便于设计。

1.1 Mesh Subdivision（up-sampling, 网格细分）

1.1.1 Loop Subdivision

命名来自于发明者的家族姓名，与loop的本意无关！

基本流程：先细分，再调整

1.三角形分解成4部分

2.对每个分解块调整位置

New Vertices（某一条新边的中点）

该点（图中为白点）必为两个三角形共享。


$$
Update=\frac{3}{8}(A+B)+\frac{1}{8}(C+D)
$$
即新顶点的值是周围顶点的加权平均，让面变得平滑。

Old Vertices


$$
n=Vertex-degree
$$

$$
u=\begin{cases}
\frac{3}{16},n=3\
\frac{3}{8n},otherwise
\end{cases}
$$

$$
Update=(1 - n*u) * original-position +
u * neighbor-position-sum
$$

理解：顶点的度决定顶点的“重要程度”，度数越高越应当保留其原本的位置，反之则要根据其连接顶点的位置决定其位置。
